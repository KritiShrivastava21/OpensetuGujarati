ઓપનસેતુ તમને તમારી માતૃભાષામાં ઇન્ટરનેટ ઍક્સેસ કરવાની મંજૂરી આપે છે. કોઈ એપ્લિકેશનની જરૂર નથી.

બસ તમારા બ્રાઉઝર સેટિંગ્સ બદલો અને ધમાલ મચાવો, વેબ તમારી ભાષા બોલે છે. તે ઈન્ફ્રાસ્ટ્રક્ચર છે જે તમને વેબના ભાષા-વિશિષ્ટ સંસ્કરણો સાથે જોડે છે.

ઉપયોગ શરૂ કરો: pranam.opensetu.in

ઓપનસેતુ શું કરે છે?

ઓપનસેતુ એક DNS રિઝોલ્વર છે જે તમને તમારી માતૃભાષામાં જાહેર વસ્તુઓ અને સરકારી ઉપયોગિતાઓ ઍક્સેસ કરવાની મંજૂરી આપે છે. ફક્ત તમારા ડોમેનનું નામ તમારી પોતાની ભાષામાં લખો અને તે તેને સત્તાવાર સાઇટના ભાષા સંસ્કરણ પર રીડાયરેક્ટ કરશે.

ઉદાહરણ તરીકે, જો તમે હિન્દી ભાષી છો, તો તમે દેવનાગરીમાં ફક્ત પીએમકિસાન.સર.ભા લખીને pmkisan.gov.in પર જઈ શકો છો અથવા જો તમે તમિલ ભાષાનો ઉપયોગ કરો છો તો ફક્ત பிஎம்கிசான்.ப વગેરે લખી શકો છો.

અસ્વીકરણ: OpenSetu, હાલમાં, ફક્ત URL દ્વારા ભાષા સંસ્કરણોને સપોર્ટ કરતી વેબસાઇટ્સ પર જ રીડાયરેક્ટ કરે છે. જો કોઈ વેબસાઇટ ભાષા સ્વિચિંગનું સંચાલન કરવા માટે JavaScript અથવા કૂકીઝનો ઉપયોગ કરે છે, તો વપરાશકર્તાઓએ સાઇટના ભાષા પસંદગીકારનો ઉપયોગ કરીને મેન્યુઅલી ભાષા બદલવાની જરૂર પડશે.

અહીં કોઈ અંગ્રેજી બાઉન્સર નથી. આ ભારતનું પોતાનું ઇન્ટરનેટ છે, જ્યાં:

  • તમારી ભાષા ડિફોલ્ટ છે, અપવાદ નથી.
  • જૂનું ઇન્ટરનેટ પહેલાની જેમ જ કામ કરે છે.
  • હવે તમારી મૂળ સ્ક્રિપ્ટમાં વેબસાઇટ્સ ઍક્સેસ કરો.
  • દરેક નાગરિકને ડિજિટલ જાહેર ચીજવસ્તુઓની ઍક્સેસ મળે છે.
