દસ્તાવેજીકરણ

ઓપનસેતુ: કેવી રીતે કાર્ય કરે છે

ઓપનસેતુ DNS over HTTPS (DOH) દ્વારા ચાલે છે, એક નેટવર્ક પ્રોટોકોલ જે ડોમેન નામોને HTTPS પર સુરક્ષિત રીતે રિઝોલ્વ કરે છે. DOH ગોપનીયતા અને સુરક્ષાને વધારશે કારણ કે મધ્યવર્તીઓને DNS ક્વેરીઝ પર ચોરવવા અથવા ફેરફાર કરવા આપતું નથી.

ભારત કા ઇન્ટરનેટ

ભારત કા ઇન્ટરનેટ Handshake પર બનાવાયું છે, એક ઓપન-સોર્સ, વિકેન્દ્રિત નેમિંગ પ્રોટોકોલ જે ઇન્ટરનેટ માટે વિકલ્પ રુટ ઝોન તરીકે સેવા આપે છે. તે સેન્સરશિપ-પ્રતિરોધી, ફેરફાર-પ્રૂફ ડોમેન રિઝોલ્યુશન સક્ષમ કરે છે. અમારી ઇન્ટરનેટનો રીડ મોડી.

નજીકના ભવિષ્યમાં, અમે ENS (Ethereum Name Service) અને IPFS (InterPlanetary File System) જેવા પ્રોટોકોલ્સને સપોર્ટ કરવાનો આયોજન પણ કરીએ છીએ, જેથી વધુ ખુલ્લા અને ઇન્ટરઓપેરેબલ વેબને સક્ષમ બનાવી શકાય.

તમારો પોતાનો DOH બનાવો

કોઈપણ વ્યક્તિ DOH રિઝોલ્વર બનાવીને ભારતીય ઇન્ટરનેટ ઍક્સેસ અને વિસ્તૃત કરી શકે છે, હોતું તો તે શાળા, યુનિવર્સિટી, સ્ટાર્ટઅપ, નોન-પ્રોફિટ, કે સરકારી એજન્સી હોય. ઓપનસેતુ માત્ર એક ઉદાહરણ છે; તમે તમારો પોતાનો ચલાવી શકો છો.
